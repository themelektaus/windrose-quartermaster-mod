Custom Ship Shanties

Replace any of the 10 vanilla sea-shanties the crew sings on board (Cutter, Brig, Ketch, Frigate; player or crew-only). A replacement applies to all ship sizes automatically.

Drop an audio file per slot - Quartermaster transcodes it to a 44.1 kHz stereo WAV (if needed), encodes it to Bink Audio and splices the result into a SoundWave template at build time. No UE5 Editor cook needed.

Accepted formats: .wav, .mp3, .ogg, .flac, .m4a, .aac, .opus. Non-WAV files are transcoded by ffmpeg.exe (~190 MB) which is staged during first-time setup. If you skipped that step, re-open setup to enable non-WAV uploads.
