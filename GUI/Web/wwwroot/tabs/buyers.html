Notes

Each entry is editable: type a new item id (autocompletes from every vanilla item), change quantities, or click the X to remove an entry from the list. The reset arrow reverts a single edit to its vanilla baseline; the + Add entry button appends a brand-new custom recipe. All edits live on the current profile and are baked into the pak on Build.
