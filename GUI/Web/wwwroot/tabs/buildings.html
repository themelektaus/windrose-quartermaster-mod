Building Creator

Add brand-new buildables to the game's build menu. Each custom building is cloned from a Vanilla "Buildable" template, with your cooked mesh / icon / image swapped in via post-cook NameMap rewrites.

How it works

Cook your assets in the UE editor first (mesh + icon + textures + MIs). Point the building at the cooked folder and pick a template - the GUI reads the mesh, lists its material slots, and lets you pick a Vanilla MI parent per slot. Picking a parent unlocks param controls for whatever scalars/vectors/textures the MI exposes; if your cooked MI uses the same parent master we pre-fill its values as defaults. User-cooked MIs/Materials get skipped at build time (they crash shipping) - cloned Vanilla MIs take their place via NameMap rewrites.
