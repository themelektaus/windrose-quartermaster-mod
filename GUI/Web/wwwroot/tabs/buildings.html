Building Creator

Add brand-new buildables to the game's build menu. Each custom building is cloned from a Vanilla "Buildable" template, with your cooked mesh / icon / image swapped in via post-cook NameMap rewrites.

How it works

Cook your assets in the UE editor first (mesh + icon + Albedo image), then point the building at the cooked folder. The build skips user-cooked Materials/MIs (they crash shipping) and replaces them with Vanilla-MI clones - your textures take over via NameMap rewrites.
