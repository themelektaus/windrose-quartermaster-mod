Setup

Re-open the first-run setup dialog to re-dump vanilla item JSONs or re-extract icon textures - useful after a game update.

Export

Extracts every building asset (DataAssets, Blueprints, Static Meshes, Materials, Textures, audio cues) from the vanilla IoStore container to Sources/Vanilla/R5/Content/ so you can clone them in the UE editor.

Heads up: writes several hundred MB to disk and takes 1-3 minutes the first time. Re-running is incremental - existing files are kept.
