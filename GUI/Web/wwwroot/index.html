First-time setup

The configurator needs to extract the vanilla item JSONs and icon textures from your Windrose install before it can show you anything. This takes 30-90 seconds the first time and is cached afterwards.

    No profiles yet
    
        
    
          Quartermaster is profile-driven: every build configuration - stack
          sizes, loot multipliers, pickup radius, fast-travel caps, ...
          lives inside a profile so you can keep multiple presets side by
          side and switch between them without losing tweaks.
        
    
        
    Create your first profile to get started.

        No Smoke
        
            
        
              Hide the smoke / flame Niagara FX on selected craft stations and campsites.
              Self-baked from vanilla: every EmitterHandle on the matching NiagaraSystem is flipped to disabled,
              then re-packed into the shared IoStore triplet.

        Notes
        
          
        
            Deleting a Quartermaster mod sends it to the Windows recycle bin - you can restore it from Explorer if needed.
            Foreign mods are read-only; manage them with whatever tool installed them.
