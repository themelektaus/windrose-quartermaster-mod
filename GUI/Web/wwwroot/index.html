        Globals - Pickup Radius
        
          
        Auto-pickup magnet range. Vanilla: 4 m.
        
          
        
            
            
        
              
              
        
                Multiplier: 2.0×
                ·
                Range: 8.0 m
              
        
            
        
          
        
          
        Ships as an IoStore triplet (.pak + .ucas + .utoc) sharing the main pak's basename, so the engine mounts everything as one logical mod. Built-in profiles can't toggle this -- duplicate first.
        
        
        

        
        
          
        Globals - Fast-Travel Bells
        
          
        Placement caps for fast-travel bells and signal fires. Vanilla: 10 bells (per type) + 3 signal fires.
        
          
        
            
            
          
        
          
        Both bell variants (DA_BI_Utilities_FastTravel_Bell + _Bell_02) share the same cap. Patches the small DA_BuildLimits_FastTravel.json shipped inside the main pak -- no extra files. Built-in profiles can't change this -- duplicate first.
        
        
        

        
        
          
        Globals - Building Stability
        
          
        Buildings stop becoming unstable: structures hold cantilevers and tall towers without collapsing. Bakes the BetterStructureSupport reference mod's 787 DA_BI* DataAssets into the IoStore triplet alongside any other globals.
        
          
          
        Single toggle: on bundles fixed values (BlockWeight=0, MaxLoad=10 000 000, MinIntersection=0); off ships nothing. Built-in profiles can't toggle this -- duplicate first.
