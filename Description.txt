[size=4][b]Description[/b][/size]

Quartermaster is a profile-driven configurator and build pipeline for Windrose. A small desktop GUI lets you mix and match gameplay tweaks across items, loot, building, navigation and FX, then bakes them into a single [code]_P.pak[/code] you drop into the [code]~mods[/code] folder. Every value is calculated against a live vanilla snapshot extracted directly from the game's pak, so multipliers really do what they say.

There is no UE4SS / SML dependency. Everything ships as a pure data pak that works in singleplayer, on dedicated servers, and on co-op sessions as long as the same pak is installed on both sides.


[size=4][b]Main features[/b][/size]

[list]
[*][b]Stack sizes[/b] - per-item or global multiplier/absolute caps; weapons, armor and other unstackable items stay vanilla unless you explicitly promote them.
[*][b]Loot tables[/b] - per-category Min/Max multipliers (mobs, plants, chests, fishing, ...) leave everything else alone.
[*][b]Pickup radius[/b] - free slider from 1.0x to 10.0x for the auto-pickup magnet range (vanilla 4 m).
[*][b]Fast-travel bells & signal fires[/b] - raise the placement caps (vanilla 10 / 3).
[*][b]Building stability[/b] - structures hold longer cantilevers and taller towers without collapsing.
[*][b]Minimap range[/b] - 1.0x to 5.0x slider for the foot/ship reveal range (vanilla 2.5 m / 7.5 m).
[*][b]Bonfire radius[/b] - 1.0x to 5.0x slider for the building-center influence sphere (vanilla 50 m / 30 m).
[*][b]No smoke[/b] - hide the smoke and flame Niagara FX on campfires/bonfires, furnaces or kilns.
[*][b]Mods tab[/b] - lists every [code].pak[/code] in your [code]~mods[/code] folder, marks Quartermaster builds, and lets you recycle-bin them with one click.
[/list]

Each feature has its own toggle and (where applicable) slider, so you can ship a pak that only changes pickup radius, or one that touches half the game - same pipeline either way.


[size=4][b]Installation[/b][/size]

[list]
[*][b]1. [/b]Download and run [code]Quartermaster.exe[/code]. On first start it auto-extracts the vanilla item JSONs and icons from your Windrose install - takes 30-90 seconds.
[*][b]2. [/b]Create a profile, pick the tweaks you want, hit [b]Build[/b]. The resulting [code]_P.pak[/code] lands directly in [code]<Steam>\steamapps\common\Windrose\R5\Content\Paks\~mods\[/code].
[*][b]3. [/b]Launch the game (or restart the server). For multiplayer the same pak has to live in the server's [code]~mods[/code] folder.
[*][b]4. [/b]To switch between configurations, build a new profile and let the Mods tab recycle the old pak - only one Quartermaster build should live in [code]~mods[/code] at a time.
[/list]


[size=4][b]Requirements[/b][/size]

[list]
[*][b]Windrose[/b] (any version that supports the [code]~mods[/code] pak loading convention).
[*][b]Microsoft Edge WebView2 runtime[/b] - preinstalled on Windows 11 and recent Windows 10 builds.
[*]No UE4SS, no SML, no other mod loader required.
[/list]
